# MushkilPay frontend

Vite + React app for the MushkilPay wallet.

See the repository **README.md** (project root) for Oracle backend setup, `.env`, and how to run API + UI together.

```bash
npm install
npm run dev
```

Uses `VITE_API_BASE_URL` (empty in dev) with Vite proxying API routes to FastAPI on port 8000.
